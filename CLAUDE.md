# CLAUDE.md - EEZO 価格帯ジャンプ効果シミュレーター

## プロジェクト概要

- **目的**: EEZO ECサイトで10,000円帯の顧客を20,000円帯へジャンプさせる商品構造設計のシミュレーション
- **背景**: Shopifyリニューアルに向けた3段階価格帯（5,000円/10,000円/20,000円）の企画
- **核心的な問い**: 「20,000円で送料無料+α」は10,000円帯顧客のジャンプを促進するか？
- **評価指標**: 
  - ジャンプ率（10,000円帯→20,000円帯への移行率）
  - AOV（平均注文額）の変化
  - 売上総額・粗利総額への影響

---

## Core Philosophy（基本原則）

### 1. Hypothesis-Driven（仮説駆動）
シミュレーションは明確な仮説から始める。「なんとなく」のパラメータ設定は禁止。

### 2. Evidence-Based（エビデンスベース）
パラメータは行動経済学の研究やEC業界データに基づく。根拠なき数値は使わない。

### 3. Sensitivity-Aware（感度意識）
不確実なパラメータは感度分析で影響度を確認する。

### 4. Business-Oriented（事業視点）
最終的に「やるべきか、やらざるべきか」の意思決定に資する出力を行う。

## ドメイン知識（EEZO固有）

### 現状の顧客構成（推定）

| 価格帯 | 構成比 | 平均単価 | 顧客像 |
|--------|--------|----------|--------|
| ～5,000円 | 10% | 3,500円 | 試し買い |
| 5,000～10,000円 | 35% | 7,500円 | ライトユーザー |
| 10,000～15,000円 | 30% | 12,500円 | **ジャンプ候補（中核）** |
| 15,000～20,000円 | 15% | 17,500円 | **ジャンプ候補** |
| 20,000円～ | 10% | 28,000円 | ヘビーユーザー |

### 閾値ジャンプ効果のエビデンス

| 出典 | 主要発見 |
|------|---------|
| Shopify調査 | 58%が送料無料のため追加購入を検討 |
| 同上 | 80%が最低購入額を満たそうとする |
| 業界調査 | 送料無料閾値でAOV 30%増 |
| Statista | 39%が送料でカート離脱 |

### 心理メカニズム

1. **Mental Accounting**: 送料は「損失」、商品追加は「利得」と認識
2. **Loss Aversion**: 「あと○○円で送料無料」を失いたくない
3. **Goal Gradient Effect**: 閾値に近いほど達成意欲が高まる
4. **Endowment Effect**: 送料無料を「すでに自分のもの」と感じる
